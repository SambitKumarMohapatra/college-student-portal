College Student Portal — LMS Backend                    
Landmine Soft (Software Developer Assignment)
Live: https://college-student-portal-9mbk.onrender.com/swagger-ui/index.html
GitHub: https://github.com/SambitKumarMohapatra/college-student-portal

- Built a production-ready REST API for college management using
  Java 17, Spring Boot 3.2, Spring Security, and PostgreSQL
- Implemented JWT authentication with access token (24hr) + refresh
  token (7 days) and token blacklisting on logout
- Designed Role-Based Access Control (RBAC) for 3 roles —
  Student, Faculty, and Admin with protected endpoints
- Integrated Gmail SMTP for password reset via email with
  time-expiring tokens stored in PostgreSQL
- Dockerized the application and deployed on Render cloud with
  PostgreSQL as managed database
- Documented all 20+ APIs using Swagger/OpenAPI 3 with
  request/response examples
